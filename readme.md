# Windows 10 Explorer & Taskbar Acrylic Transparency Tweak

Skrip ini mengaktifkan efek transparansi Acrylic pada Windows Explorer dan Taskbar melalui modifikasi Registry tanpa memerlukan aplikasi pihak ketiga.

## 🚀 Fitur
- Mengaktifkan `AcrylicTransparency` pada tingkat sistem.
- Menyesuaikan pengaturan `Advanced` pada Explorer.
- Mengatur izin Registry secara otomatis agar perubahan tetap permanen.

## 🛠 Cara Penggunaan
1. Buka **PowerShell** dengan hak akses **Administrator**.
2. Salin dan tempel kode di bawah (lihat bagian skrip).
3. Restart **Windows Explorer** atau **Log out** untuk melihat hasilnya.

## ⚠️ Catatan
Tweak ini bekerja paling baik pada Windows 10 versi 1903 ke atas (termasuk Pro dan Enterprise). Efek mungkin bervariasi tergantung pada konfigurasi GPU dan pengaturan Power Plan (High Performance disarankan).
